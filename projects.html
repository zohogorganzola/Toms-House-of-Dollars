HoverBoard

A .Net/C# App I wrote a few years ago with the intention of adding global shortcut commands to modifying the contents of your clipboard. Got as far as a few whitespace/capitalization behaviors and a custom regex option. The plan was to eventually have shortcuts to handle a set of supported data, like pictures or sound. Shallow features in the app (mainly cropping) with a button push the data as a file stream to an associated editor. But for the time being my focus is on web apps.
